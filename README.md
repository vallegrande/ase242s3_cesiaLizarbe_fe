# Proyecto Angular Hola Mundo

## Descripción
Este proyecto es una **aplicación básica de Angular** que muestra el mensaje **"Hola Mundo"** en pantalla usando un **componente personalizado** llamado `saludo`.  
El objetivo es aprender a crear un proyecto Angular desde cero, manejar componentes y verificar que Node, npm y Angular CLI estén correctamente instalados.

---

## ¿Qué es Angular?
Angular es un **framework de JavaScript** que permite crear aplicaciones web modernas y dinámicas.  
Se basa en **componentes**, que son partes independientes de la interfaz que se pueden reutilizar y combinar para formar la aplicación completa.

---

## Herramientas necesarias
Para poder desarrollar este proyecto, se necesitan instalar las siguientes herramientas:

1. **Node.js** – Plataforma que permite ejecutar JavaScript fuera del navegador.  
   Versión usada: `22.14.0`

2. **npm (Node Package Manager)** – Administrador de paquetes que permite instalar librerías y dependencias.  
   Versión usada: `10.9.2`

3. **Angular CLI** – Interfaz de línea de comandos de Angular para crear y gestionar proyectos Angular.  
   Versión usada: `19.2.11`
